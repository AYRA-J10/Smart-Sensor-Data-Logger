# Smart Sensor Data Logger

## Overview

This project simulates a noisy temperature sensor using Python.

The sensor readings are generated with random noise using NumPy.

A simple Kalman-style smoothing algorithm is applied to reduce noise.

Every reading is saved into a CSV file.

If the temperature exceeds the threshold, a warning is displayed.

Finally, the raw and smoothed sensor readings are plotted using Matplotlib.

## Technologies

- Python
- NumPy
- Matplotlib
- CSV

## Features

- Simulated sensor
- Noise generation
- Data smoothing
- Threshold alert
- CSV logging
- Graph plotting
